This is a Vite + React portfolio with a Three.js hero and parallax scroll effects.

## Quick start

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start dev server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

## Deployment

- GitHub Pages: build and push `dist/` to `gh-pages` branch or configure GitHub Actions.
- VPS: upload the `dist/` folder to your web server.
